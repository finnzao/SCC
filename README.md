# SCC — Sistema de Controle de Comparecimento


## O que faz

- Cadastro e acompanhamento
- Registro de comparecimentos (presencial ou balcão virtual) com verificação de endereço
- Controle automático de prazos, periodicidade e inadimplência
- Dashboard com visão geral: estatísticas, gráficos de conformidade, análise de atrasos por faixa (30/60/90/90+ dias)
- Alertas de custodiados atrasados e urgentes
- Gestão de usuários por convite com perfis Admin e Usuário
- Exportação de relatórios em Excel
